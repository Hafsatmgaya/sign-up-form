Title:
90-Day Launch Circle — Sign-Up Form
Description:
A sign-up form page built for the 90-Day Launch Circle, a coaching program designed for 9-to-5 workers ready to launch their own business. This project was completed as part of The Odin Project's Intermediate HTML and CSS curriculum.
What I practiced:
Two-column layout using Flexbox
Form structure with semantic HTML
CSS Grid for the two-column input layout
Pseudo-classes — :focus for active input styling and :invalid for email validation feedback
Background images and CSS positioning
Box shadows and border styling

Built with:
HTML5
CSS3
Google Fonts — Roboto
